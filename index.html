Prework Study Guide

HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • These elements are, respectively, the basis of the front end and back end of a webpage.
  • HTML provides the content of a webpage.

CSS

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • CSS adds style(s) to the webpage content provided by the HTML section

Git

  • Git Flow is the process through which multiple users can work on a branch of code at the same time.
  • Developers can use the pull and push processes to create new branches of existing code for them to work on.
  • Git Flow demonstrates the agile development strategy by allowing other developers to review your branch that is a work in progress.
  • Git ensures that your changes are added to whatever project you are working on either locally or on the network.

JavaScript

  • Javascript uses logic to add functionality to the stylized webpage.
  • The functionality allows the user to interact with the webpage.
